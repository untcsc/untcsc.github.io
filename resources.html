Kali Linux

An Advanced Penetration Testing Linux distribution used for Penetration Testing, Ethical Hacking and network security assessments.

Ubuntu

ubuntu.com

Ubuntu is an open source (Debian Linux) operating system that runs from the desktop, to the cloud, and to all your internet connected things.

Wireshark

wireshark.org

Wireshark is the world's foremost and widely-used network protocol analyzer. It lets you see what's happening on your network at a microscopic level.

VirtualBox

virtualbox.org

VirtualBox is a powerful x86 and AMD64/Intel64 virtualization product for enterprise as well as home use.

Putty

WinSCP

winscp.net

WinSCP is a popular SFTP client and FTP client for Microsoft Windows! Copy file between a local computer and remote servers using FTP, FTPS, SCP, SFTP, WebDAV or S3 file transfer protocols.

Metasploit

metasploit.com

The Metasploit Framework contains a suite of tools that you can use to test security vulnerabilities, enumerate networks, execute attacks, and evade detection.

Cryptool

cryptool.org

CrypTool is an open-source Windows program for cryptography and cryptanalysis. It's the most wide-spreaded e-learning software of its kind.

HxD

HxD is a carefully designed and fast hex editor which, additionally to raw disk editing and modifying of main memory (RAM), handles files of any size.
